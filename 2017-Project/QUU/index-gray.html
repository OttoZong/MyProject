心路歷程

  • 2016年3月

    決定以「大學T」作為主題進行專題的發展

  • 2016年4月

    尋找適合的指導老師

    我們在找尋指導老師時第一時間就認為歐陽芳泉老師最為合適，在多媒體的領域能給予專業的的知識和良好建議。

  • 2016年5月

    規劃專題發展項目細節

    製作項目規劃出來平面設計：衣服（實體商品）、宣傳海報，網站預計製作宣傳及形象網站的模式，影片設計，預計製作一個置入式一則故事短片當中放入我們的商品。

  • 2016年6月

    尋找廠商，洽談合作相關事宜

  • 2016年9月

    開始執行專題計畫

    從平面為起點，開始執行第一步，接著細部分工網站設計與影片設計，每個人負責擅長的領域，著手完成。

  • 2017年1月

    網站雛型完成

    完成網站地圖，網站內容與資料的收集彙整。

  • 2017年2月

    平面設計雛型完成

    將平面想法產出，進行細部調整作業。

  • 2017年3月

    微電影腳本完成，並開始進行拍攝

    進行微電影腳本撰寫。

  • 2017年4月

    平面打版、微電影拍攝完成

    衣服成品出產，影片進入後製剪輯階段。

  • 2017年5月

    平面拍攝、微電影等作品彙整至網站

    拍攝平面類照片，完成海報輸出，影片匯入網站方便進行觀賞，並做最後的微調。
